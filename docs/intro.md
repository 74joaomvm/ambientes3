# Introdução ao Tutorial

Vamos descobrir como criar um site para **gestão de tabacaria** com **Docusaurus em menos de 5 minutos**.

## Começando

Comece criando um novo site para sua **tabacaria**.

Ou **experimente o Docusaurus imediatamente** com **[docusaurus.new](https://docusaurus.new)**.

### O que você vai precisar

- [Node.js](https://nodejs.org/en/download/) versão 20.0 ou superior:
  - Ao instalar o Node.js, é recomendado que você marque todas as caixas relacionadas às dependências.

## Gerar um novo site

Gere um novo site Docusaurus utilizando o **modelo clássico**.

O modelo clássico será automaticamente adicionado ao seu projeto após você rodar o comando:

```bash
npm init docusaurus@latest meu-site-tabacaria classic
Você pode digitar esse comando no Prompt de Comando, Powershell, Terminal, ou em qualquer outro terminal integrado do seu editor de código.

Esse comando também instala todas as dependências necessárias para você rodar o Docusaurus.
