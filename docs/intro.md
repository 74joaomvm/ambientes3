# Tutorial de Uso da Aplicação de Gestão de Tabacaria

Bem-vindo ao **Tutorial de Gestão da Tabacaria**! Neste tutorial, vamos guiá-lo pelas funcionalidades da nossa aplicação para gestão de tabacarias. A aplicação foi criada para facilitar o gerenciamento de produtos, promoções, estoque e pedidos.
